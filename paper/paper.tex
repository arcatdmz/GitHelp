\begin{abstract}
  We introduce a flexible command translation system that can generate
  a complex complete command string from vague keywords
  given by the user.
  %
  % Although intuitive GUI are 
  % command-line interface (CLI) is stil widely used everywhere,
  %
  People use computers for various tasks that they want to perfer,
  but the language and vocabularies used for the task is usually completely different.
  When a user wants to ``make the room darker'',
  he should translate it to the real action like ``turn off the ceiling light'',
  and toggle the power switch or invoke a command like ``\texttt{\% iotctl ceilinglight 0}''.
  If the user is a Chinese speaker, he might have to translate his mind
  % ``使房黑暗'' into ``make the room darker'', and finally to \texttt{\% iotctl ceilinglight 0}.
  ``使房黒暗'' into ``make the room darker'', and finally to ``\texttt{\% iotctl ceilinglight 0}''.

  People in the world are always suffering from this kind of translation problem.
  Translation problems are serious for experienced Western computer users, but
  they are more serious for vast amount of people around the world.

  We propose a simple and general translation framework
  ``ExpandHelp'' that can be used in wide area of situation
  where translation is required, and
  show how the framework can be used
  by showing an example system ``\texttt{GitHelp}''
  that can help \texttt{git} users.

\end{abstract}
